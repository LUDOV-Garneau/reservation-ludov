# Guide de consultation de vos réservations

Ce guide explique comment consulter vos réservations à venir et votre historique sur la plateforme LUDOV. Pour savoir comment effectuer une nouvelle réservation, consultez notre guide dédié : [Effectuer une réservation](/docs?page=effectuer_reservation&adminRessources=false).

---

## Page d'accueil

Sur la page d'accueil, vous trouverez deux sections principales :

1.  **Réservations à venir** : Affiche vos réservations futures confirmées.
2.  **Historique** : Affiche vos réservations passées ainsi que celles qui ont été annulées.

![Réservations à venir](/images/tutoriels/consultation_reservations/reservations-a-venir.png)

![Historique des réservations](/images/tutoriels/consultation_reservations/historique-reservations.png)

Les réservations annulées apparaissent également dans l'historique avec un statut distinct.

![Historique avec réservation annulée](/images/tutoriels/consultation_reservations/historique-reservation-annulee.png)

## Détails d'une réservation

Pour voir les détails d'une réservation, cliquez sur le bouton **"Détails"** pour la réservation que vous souhaitez consulter.

### En-tête et options

En haut de la page de détails, vous trouverez les informations principales (**Date**, **Heure**, **Plateforme**) ainsi que plusieurs options :

-   **Ajouter au calendrier** : Télécharge un fichier `.ics` pour ajouter la réservation à votre calendrier personnel.
-   **Rappels par courriel** : Permet de configurer des alertes pour ne pas oublier votre réservation. [Voir comment activer les rappels](/docs?page=rappels&adminRessources=false).
-   **Annuler la réservation** : Permet d'annuler votre réservation si vous ne pouvez plus vous présenter.

![En-tête des détails](/images/tutoriels/consultation_reservations/details-reservation-entete.png)

### Informations de la réservation

La page de détails liste également tous les éléments inclus dans votre réservation :

-   La **console** réservée.
-   Les **jeux** sélectionnés.
-   Les **accessoires** choisis.

![Détails complets](/images/tutoriels/consultation_reservations/details-reservation.png)

## Annuler une réservation

Si vous devez annuler une réservation, cliquez sur le bouton **"Annuler la réservation"** dans la page de détails. Une confirmation vous sera demandée.

![Annulation de réservation](/images/tutoriels/consultation_reservations/annuler-reservation.png)
