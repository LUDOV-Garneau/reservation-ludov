# Guide de réservation d'une station de jeu

Ce guide explique comment effectuer une réservation de station de jeu sur la plateforme LUDOV.

LUDOV dispose d'un local équipé de stations (TV HD, Casque VR, etc.) que vous pouvez réserver pour des blocs de 2 heures afin de jouer à un jeu vidéo sur une console spécifique.

---

## Processus de réservation

Suivez ces étapes pour réserver une station de jeu.

### Étape 1 : Sélection de la console

La première étape consiste à choisir la console que vous souhaitez utiliser.

- Une **barre de recherche** est disponible pour trouver rapidement une console.
- Seules les consoles disponibles dans le local sont affichées.

![Sélection de la console](/images/tutoriels/effectuer_reservation/console.png)

Vous pouvez voir les détails de la console avant de la sélectionner.

![Détails de la console](/images/tutoriels/effectuer_reservation/console-selection.png)

> **Important** : Dès que vous sélectionnez une console, vous disposez de **15 minutes** pour compléter votre réservation. Un minuteur s'affichera pour vous indiquer le temps restant.
>
> Il est possible d'annuler le processus de réservation en tout temps si vous changez d'avis.

### Étape 2 : Sélection des jeux

Une fois la console choisie, vous devez sélectionner les jeux auxquels vous souhaitez jouer.

- Seuls les jeux compatibles avec la console sélectionnée sont affichés.
- Vous pouvez utiliser la **barre de recherche** pour trouver un jeu spécifique.
- Vous pouvez sélectionner jusqu'à **3 jeux** maximum.

![Liste des jeux](/images/tutoriels/effectuer_reservation/jeux.png)

![Sélection des jeux](/images/tutoriels/effectuer_reservation/jeux-selection.png)

### Étape 3 : Sélection des accessoires

Vous pouvez ensuite ajouter des accessoires à votre réservation.

- Seuls les accessoires compatibles avec la console sélectionnée sont affichés.
- Une **barre de recherche** est disponible.
- Il n'y a **pas de limite** sur la quantité d'accessoires que vous pouvez réserver (dans la limite des stocks disponibles).

> **Note** : Pour certains jeux, un accessoire peut être requis. Dans ce cas, il sera **pré-sélectionné** automatiquement dans cette page.

![Liste des accessoires](/images/tutoriels/effectuer_reservation/accessoires.png)

![Sélection des accessoires](/images/tutoriels/effectuer_reservation/accessoires-selection.png)

### Étape 4 : Choix de la date et de l'heure

Vous devez maintenant choisir le moment de votre réservation.

- Les plages horaires valides sont définies par un administrateur.
- Les réservations se font toujours par **blocs de 2 heures**.
- Seules les heures et dates pour lesquelles la console, les jeux et les accessoires sélectionnés sont tous disponibles sont affichées.

![Calendrier](/images/tutoriels/effectuer_reservation/date.png)

![Sélection de l'heure](/images/tutoriels/effectuer_reservation/date-selection.png)

> **Restrictions de réservation** :
> - Vous ne pouvez pas réserver pour le **jour même** ou le **lendemain**.
> - Vous ne pouvez pas réserver la **même console** deux fois dans la **même journée**.

### Étape 5 : Sélection du cours

Vous devez indiquer pour quel cours universitaire vous effectuez cette réservation.

- Sélectionnez votre cours dans la liste.
- Si votre cours n'est pas listé ou si la réservation est pour un autre motif académique, vous pouvez sélectionner l'option **"Autre"**.

![Sélection du cours](/images/tutoriels/effectuer_reservation/cours.png)

![Cours sélectionné](/images/tutoriels/effectuer_reservation/cours-selection.png)

### Étape 6 : Confirmation

Une page de détails s'affiche pour récapituler tous vos choix (console, jeux, accessoires, date, heure, cours).

Vérifiez attentivement les informations avant de confirmer.

![Confirmation de la réservation](/images/tutoriels/effectuer_reservation/confirmation.png)

### Étape 7 : Réservation confirmée

Une fois la réservation confirmée :
1. Elle s'affichera sur votre **tableau de bord**.
2. Vous recevrez un **courriel de confirmation**.

![Réservation confirmée](/images/tutoriels/effectuer_reservation/confirmee.png)
