# Tutoriel : Système de Politique dans le Panneau Admin

## Introduction

Ce tutoriel vous guide dans l'utilisation du système de politique disponible dans le panneau d'administration.

## Accès au Système de Politique

1. Connectez-vous au panneau d'administration
2. Naviguez vers la section **Politiques**

## Création d'une Nouvelle Politique

1. Cliquez sur le bouton **"Nouvelle Politique"**
2. Remplissez les champs requis :
   - **Nom** : Nom de la politique
   - **Description** : Description détaillée
   - **Type** : Sélectionnez le type de politique
3. Cliquez sur **"Enregistrer"**

## Modification d'une Politique

1. Trouvez la politique dans la liste
2. Cliquez sur l'icône **Éditer** (crayon)
3. Modifiez les informations nécessaires
4. Cliquez sur **"Sauvegarder les modifications"**

## Activation/Désactivation

- Utilisez le bouton bascule à côté de chaque politique
- Une politique active est indiquée par un statut vert
- Une politique inactive est indiquée par un statut gris

## Suppression d'une Politique

1. Cliquez sur l'icône **Supprimer** (poubelle)
2. Confirmez la suppression dans la boîte de dialogue

## Bonnes Pratiques

- Testez toujours les politiques avant de les activer
- Documentez les changements effectués
- Vérifiez les permissions associées
