# Tutoriel : Gestion des Utilisateurs - Panneau Admin

## Vue d'ensemble

Ce guide explique comment gérer les utilisateurs depuis le panneau d'administration.

## Accès au module

1. Connectez-vous au panneau admin
2. Cliquez sur **Utilisateurs** dans le menu principal

## Créer un utilisateur

1. Cliquez sur le bouton **+ Nouvel utilisateur**
2. Remplissez les champs obligatoires :
   - **Nom complet**
   - **Email** (doit être unique)
   - **Mot de passe** (minimum 8 caractères)
   - **Rôle** (Administrateur, Gestionnaire, Utilisateur)
3. Cliquez sur **Enregistrer**

## Modifier un utilisateur

1. Recherchez l'utilisateur dans la liste
2. Cliquez sur l'icône **✏️ Éditer**
3. Modifiez les informations souhaitées
4. Cliquez sur **Mettre à jour**

## Gérer les permissions

1. Sélectionnez un utilisateur
2. Allez dans l'onglet **Permissions**
3. Cochez/décochez les accès :
   - Lecture
   - Écriture
   - Suppression
   - Administration
4. Cliquez sur **Appliquer**

## Désactiver/Supprimer un utilisateur

### Désactivation (recommandé)

1. Cliquez sur **⋮** à côté de l'utilisateur
2. Sélectionnez **Désactiver**
3. Confirmez l'action

### Suppression définitive

1. Cliquez sur **⋮** à côté de l'utilisateur
2. Sélectionnez **Supprimer**
3. **⚠️ Attention** : Cette action est irréversible

## Réinitialiser un mot de passe

1. Sélectionnez l'utilisateur
2. Cliquez sur **Réinitialiser le mot de passe**
3. Un email sera envoyé automatiquement

## Filtres et recherche

- **Barre de recherche** : Recherche par nom ou email
- **Filtres** : Par rôle, statut (actif/inactif), date de création

## Bonnes pratiques

- ✅ Utilisez des rôles appropriés selon les besoins
- ✅ Désactivez plutôt que supprimer pour conserver l'historique
- ✅ Vérifiez régulièrement les utilisateurs inactifs
- ❌ Ne partagez jamais les identifiants admin
