# Guide de connexion - Tutoriel pas à pas

## Étape 1 : Accéder à la page de connexion

1. Ouvrez votre navigateur web
2. Entrez l'URL de l'application dans la barre d'adresse
3. Appuyez sur **Entrée**

## Étape 2 : Remplir le formulaire de connexion

1. Localisez le champ **Email** ou **Nom d'utilisateur**
2. Cliquez dans le champ et saisissez votre adresse email
3. Appuyez sur **Tab** pour passer au champ suivant
4. Saisissez votre **mot de passe**
   - Assurez-vous que votre mot de passe respecte la casse (majuscules/minuscules)

## Étape 3 : Options supplémentaires (optionnel)

- Cochez la case **"Se souvenir de moi"** si vous souhaitez rester connecté
- Cette option est recommandée uniquement sur un appareil personnel

## Étape 4 : Se connecter

1. Cliquez sur le bouton **"Se connecter"** ou **"Connexion"**
2. Patientez quelques secondes pendant la vérification de vos identifiants

## Étape 5 : Accès à l'application

- Si vos identifiants sont corrects, vous serez redirigé vers le tableau de bord
- En cas d'erreur, un message vous indiquera le problème

## Mot de passe oublié ?

1. Cliquez sur le lien **"Mot de passe oublié ?"**
2. Suivez les instructions pour réinitialiser votre mot de passe

## Problèmes courants

- **Email invalide** : Vérifiez l'orthographe de votre email
- **Mot de passe incorrect** : Vérifiez que le verrouillage majuscules n'est pas activé
- **Compte bloqué** : Contactez l'administrateur
